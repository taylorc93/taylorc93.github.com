Wavely

Wavely is an iOS application built in Swift and PureData. Wavely allows users to create their own complex sound waves through layering wave shapes. These waves can be presets or user generated shapes built using a touch enabled canvas. Wave shapes can then be played through our synth interface. Includes a customizable random note generator in addition to a simple piano interface.

This app is under heavy development. Expect a first release around mid October
